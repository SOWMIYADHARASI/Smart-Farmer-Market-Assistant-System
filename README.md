Smart Farmer Market Assistant System

The Smart Farmer Market Assistant System is a hardware-based solution designed to help farmers overcome the problem of price opacity in rural markets. Many farmers do not have access to accurate and real-time market prices for their crops, which allows middlemen to buy crops at very low prices. As a result, farmers often lose a significant portion of their potential profit.

This project aims to develop a low-cost smart device that assists farmers in making better selling decisions. The system is built using an Arduino microcontroller and integrates multiple modules such as a load cell sensor for weight measurement, GSM module for receiving market price updates via SMS, WiFi module for internet-based price information, and an LCD display to show the results.

When a farmer places the crop on the load cell sensor, the system measures the weight of the crop. The farmer can select the crop type using push buttons, and the device calculates the total selling value based on the current market price. The market price is updated through SMS alerts or internet data, ensuring that farmers receive the most recent price information.

Additionally, the system can suggest the best market to sell the crop by comparing prices from different markets. This helps farmers choose the location where they can earn the highest profit.

The Smart Farmer Market Assistant System is designed to be simple, affordable, and easy to use in rural areas. By providing transparent market information and automatic price calculation, this project helps farmers increase their income, reduce dependency on middlemen, and make informed selling decisions.
